Plugin para controlar as informações do checkout no WooCommerce, permitindo a personalização de campos e upload de arquivos pelos usuários.

== Description ==

O Custom Checkout é um plugin para WooCommerce que permite controlar as informações capturadas no checkout. Com este plugin, é possível:

* Escolher quais informações serão capturadas no checkout.
* Definir quais campos são obrigatórios.
* Adicionar campos de upload de arquivos.
* Personalizar os campos exibidos e obrigatórios através de um shortcode.
* Salvar e exibir informações adicionais e arquivos enviados no perfil do usuário.

== Installation ==

1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/custom-checkout`, ou instale o plugin diretamente pelo painel do WordPress.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Configure os campos do checkout e adicionais conforme necessário.

== Usage ==

Para exibir os campos adicionais em qualquer página ou post, use o shortcode `[custom_checkout_additional_info]`. Este shortcode exibirá um formulário onde os usuários podem enviar informações adicionais e fazer upload de arquivos.

= Como faço para adicionar campos personalizados no checkout? =
Os campos podem ser adicionados e personalizados através do código do plugin. Consulte a documentação para obter mais detalhes.

= Como faço para exibir as informações adicionais no perfil do usuário? =
As informações adicionais e os arquivos enviados são automaticamente salvos e exibidos no perfil do usuário no painel administrativo.
